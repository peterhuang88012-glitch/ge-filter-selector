# 全球燃气轮机进气过滤选型系统

一个专业的在线工程计算工具，支持全球主流燃气轮机厂家的进气过滤系统快速选型。基于 **GMRC 燃气轮机进气过滤系统指南 Release 1.0** 开发，包含完整的生命周期成本 (LCC) 分析和性能衰减模型。

## 功能特性

- **全球厂家支持**：涵盖 GE、西门子（Siemens）、三菱重工（Mitsubishi Power）、斗山（Doosan）、安萨尔多（Ansaldo）等主流厂家
- **丰富型号库**：20+ 种主流燃机型号，包含质量流量、功率、热耗率等关键参数
- **GMRC 指南集成**：基于 GMRC Release 1.0 标准，实现完整的 LCC 分析和性能衰减模型
- **生命周期成本分析**：
  - 初始投资成本
  - 年维护成本
  - 压降能耗成本
  - 性能衰减成本（年衰减率 1.5%）
  - 10 年周期总 LCC 成本
- **性能衰减预估**：根据 GMRC 标准计算年度功率衰减、效率衰减和衰减成本
- **环境污染物分类**：根据 GMRC 指南，详细规定了各环境的污染物浓度范围
- **科德宝滤芯推荐**：根据环境工况自动推荐科德宝 Viledon 系列滤芯及数量
- **实时计算**：基于工程经验公式和 GMRC 标准，实时计算过滤器数量、压降、功率损失等关键参数
- **环境修正**：考虑海拔、温度、湿度等环境因素对进气密度的影响
- **响应式设计**：支持桌面、平板和手机等多种设备访问

## 计算原理

### 核心公式

**1. 现场气压修正**（高度公式）
```
P_site = 101325 × (1 - 2.25577×10⁻⁵ × h)^5.25588
```

**2. 湿空气密度**
```
ρ = (P_d / (R_d × T)) + (P_v / (R_v × T))
```

**3. 现场质量流量修正**
```
m_site = m_ISO × (P_site / 101325) × √(288.15 / T_abs)
```

**4. 体积流量**
```
Q = (m_site / ρ) × 3600
```

**5. 过滤器压降**
```
ΔP = 12v² + 5v  (Pa)
```

**6. 功率损失**
```
ΔP_loss = P_rated × (ΔP / 100) × 0.00135  (MW)
```

**7. 性能衰减成本** (GMRC Section 3.4.2.3)
```
A = 0.5 × P × D × 365 × 24 × a × C_kwh
```
其中 D 为衰减率（%），a 为可用率，C_kwh 为电价。

**8. 生命周期成本** (GMRC Section 3.4.1)
```
LCC = NPV_init + NPV_maint + NPV_energy + NPV_degradation
```

## GMRC 指南核心内容

### 生命周期成本 (LCC) 分析

GMRC 指南强调了全生命周期成本分析的重要性。LCC 包括：

- **初始成本**：滤芯、框架、仪表、安装费用。
- **运营成本**：压降导致的功率损失、电费。
- **维护成本**：滤芯更换、清洁、维修。
- **衰减成本**：燃机性能衰减导致的效率和功率损失。
- **停机成本**：计划外停机导致的生产损失。

### 性能衰减模型

GMRC 指南规定，燃机在频繁在线/离线清洗的情况下，年衰减率为 **1.5%**。衰减包括：

- **功率衰减**：年均功率损失。
- **效率衰减**：年均效率损失。
- **热耗率增加**：导致燃料消耗增加。

### 环境分类与污染物

GMRC 指南详细规定了各环境的污染物浓度范围和特性，指导滤芯选型。

## 科德宝滤芯等级介绍

- **G3/G4**：粗效预过滤，主要去除大颗粒子
- **F5/F7/F9**：中效过滤，一般作为中间段
- **E10/E12**：高效过滤，作为终段过滤
- **Hydro**：疏水技术，特别适合海洋、热带等高湿环境
- **GTB/GTS**：脉冲反吹自洁系统，特别适合沙漠、高粉尘环境
