# 全球燃气轮机进气过滤选型系统

一个专业的在线工程计算工具，支持全球主流燃气轮机厂家的进气过滤系统快速选型。基于 **GMRC 燃气轮机进气过滤系统指南 Release 1.0** 开发，包含完整的生命周期成本 (LCC) 分析和性能衰减模型。

## 功能特性

- **全球厂家支持**：涵盖 GE、西门子（Siemens）、三菱重工（Mitsubishi Power）、斗山（Doosan）、安萨尔多（Ansaldo）、索拉（Solar）、曼恩（MAN）等主流厂家
- **海量型号库**：50+ 种主流燃机型号，包含质量流量、功率、热耗率等关键参数
- **GMRC 指南集成**：基于 GMRC Release 1.0 标准，实现完整的 LCC 分析和性能衰减模型
- **生命周期成本分析**：
  - 初始投资成本
  - 年维护成本
  - 压降能耗成本
  - 性能衰减成本（年衰减率 1.5%）
  - 10 年周期总 LCC 成本
- **性能衰减预估**：根据 GMRC 标准计算年度功率衰减、效率衰减和衰减成本
- **环境污染物分类**：根据 GMRC 指南，详细规定了各环境的污染物浓度范围
- **科德宝滤芯推荐**：根据环境工况自动推荐科德宝 Viledon 系列滤芯及数量
- **智能辅助系统**：基于环境温度自动推荐进气加热（防冰）或进气冷却（性能提升）系统
- **安全防护配置**：集成防爆门（Blow-in Doors）配置建议及安全平衡逻辑计算
- **专家决策支持**：提供 E12 ROI 分析、冗余设计建议及防爆门合规性深度分析
- **计算原理透明化**：详细展示滤芯数量、防爆门开启压力等核心参数的推导公式

## 使用方法

1. 访问网站：[https://peterhuang88012-glitch.github.io/ge-filter-selector/](https://peterhuang88012-glitch.github.io/ge-filter-selector/)
2. 选择燃机厂家及对应型号
3. 选择现场环境工况（标准、海洋、沙漠、极寒、热带、山区）
4. 系统将根据环境温度自动推荐是否开启**进气加热**或**进气冷却**辅助系统
5. 输入项目工况参数（海拔、温度、湿度、目标风速）
6. 系统自动计算并显示选型结果、滤芯推荐、性能衰减和 LCC 成本
7. 点击“查看计算原理”或“查看计算依据”了解核心参数的推导过程
