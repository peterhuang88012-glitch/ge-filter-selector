# GE 燃气轮机进气过滤选型系统

一个专业的在线工程计算工具，用于 GE 燃气轮机进气过滤系统的快速选型。

## 功能特性

- **多型号支持**：涵盖 GE 9HA.02、9HA.01、9F.04、9E.03、6B.03、LM6000 等主流燃机型号
- **实时计算**：基于工程经验公式，实时计算过滤器数量、压降、功率损失等关键参数
- **环境修正**：考虑海拔、温度、湿度等环境因素对进气密度的影响
- **智能布局**：自动计算最优的过滤器排列方案
- **尺寸估算**：提供进气墙尺寸和结构重量的参考数据
- **响应式设计**：支持桌面、平板和手机等多种设备访问

## 使用方法

1. 访问网站：[https://ge-filter-selector.github.io](https://ge-filter-selector.github.io)（或您的 GitHub Pages 地址）
2. 选择燃机型号
3. 输入项目工况参数（海拔、温度、湿度）
4. 设定目标过滤风速
5. 系统自动计算并显示选型结果

## 计算原理

### 核心公式

**1. 现场气压修正**（高度公式）
```
P_site = 101325 × (1 - 2.25577×10⁻⁵ × h)^5.25588
```

**2. 湿空气密度**
```
ρ = (P_d / (R_d × T)) + (P_v / (R_v × T))
```
其中：
- P_d：干空气分压
- P_v：水蒸气分压
- R_d = 287.05 J/(kg·K)
- R_v = 461.5 J/(kg·K)

**3. 现场质量流量修正**
```
m_site = m_ISO × (P_site / 101325) × √(288.15 / T_abs)
```

**4. 体积流量**
```
Q = (m_site / ρ) × 3600
```

**5. 过滤器压降**（F9 级高效过滤器）
```
ΔP = 12v² + 5v  (Pa)
```

**6. 功率损失**
```
ΔP_loss = P_rated × (ΔP / 100) × 0.00135  (MW)
```

## 参数说明

| 参数 | 范围 | 说明 |
|------|------|------|
| 海拔 | 0-5000 m | 项目所在地的海拔高度 |
| 温度 | -20~60 °C | 环境设计温度 |
| 湿度 | 0-100 % | 相对湿度 |
| 风速 | 2.0-3.5 m/s | 推荐 2.5-3.2 m/s，超过 3.5 m/s 系统会给出警告 |
