# 燃机进气系统防爆门（Blow-in Doors / Implosion Doors）选型指南

## 1. 定义与作用
防爆门（通常在燃机领域称为**吸入门**或**内爆门**）是一种安全保护装置。当进气过滤系统因滤芯堵塞、结冰或其他原因导致进气阻力急剧上升，使进气道内产生过大的负压时，防爆门会自动开启，允许部分未经空气过滤的空气直接进入燃机，以防止进气道结构因负压过大而发生坍塌（内爆）。

## 2. 需要增加防爆门的工况
根据行业标准和 GMRC 指南逻辑，以下工况强烈建议配置防爆门：
- **极寒/结冰环境**：滤芯极易因积雪或结冰而迅速堵塞，导致压降在短时间内飙升。
- **高粉尘/沙漠环境**：虽然有自洁系统，但在极端沙尘暴下，滤芯压降可能超过设计极限。
- **无人值守/远程监控站点**：无法及时人工干预滤芯更换的场合。
- **高价值/大型燃机**：为了保护昂贵的进气道结构和燃机本体安全。
- **海军/船舶应用**：由于空间受限且环境恶劣，常作为标准配置。

## 3. 关键技术参数
- **开启压力（Set Point）**：通常设定在 **114 mmH2O 至 406 mmH2O** 之间。常见的标准设定值为 **140 mmH2O (约 1370 Pa)**。
- **自动复位**：现代防爆门通常具备自动复位功能，当压差恢复正常后自动关闭。
- **信号反馈**：必须配备限位开关（Limit Switches），一旦开启需向控制系统（DCS）发出报警，甚至触发燃机减载或停机，因为此时进入的是未经过滤的空气。

## 4. 选型逻辑集成建议
- **判断逻辑**：如果 `环境 == 极寒` 或 `环境 == 沙漠` 或 `燃机功率 > 100MW`，则推荐配置。
- **提示信息**：在网页中增加“安全防护建议”板块，明确说明防爆门的必要性及设定压力。
