# 燃气轮机进气过滤系统滤芯数量计算依据

在进气过滤系统中，滤芯数量（Cells）的确定遵循以下核心工程逻辑：

## 1. 核心计算公式
滤芯数量 $N$ 的计算公式为：
$$N = \frac{Q_{site}}{3600 \times A_{cell} \times V_{target}}$$

### 参数物理意义：
*   **$Q_{site}$ (工况体积流量, $m^3/h$)**: 燃机在现场环境（考虑海拔、温度、湿度）下的实际进气需求。
*   **$A_{cell}$ (单只滤芯有效迎风面积, $m^2$)**: 标准 $24" \times 24"$ 滤芯的迎风面积，通常取 $0.35 m^2$。
*   **$V_{target}$ (设计目标风速, $m/s$)**: 穿过滤芯表面的空气流速。

## 2. 行业标准取值依据 (GMRC & OEM)
根据《GMRC 燃气轮机进气过滤系统指南》及主流 OEM（GE, Siemens）的设计规范：

| 过滤系统类型 | 推荐风速 $V_{target}$ (m/s) | 依据说明 |
| :--- | :--- | :--- |
| **静态过滤系统** | 2.5 - 3.5 | 平衡初投资与运行压降的最优区间。 |
| **脉冲反吹系统** | 4.0 - 6.0 | 脉冲系统允许更高风速，但需考虑清灰效率。 |
| **高效/超高效 (E12)** | 2.0 - 2.5 | 高等级滤芯阻力大，通常采用较低风速以延长寿命。 |

## 3. 选型系统中的逻辑实现
本系统（v10.2）采用 **3.0 m/s** 作为基准设计风速，这是行业内公认的“黄金平衡点”：
1.  **压降控制**：在此风速下，洁净滤芯的初阻力通常在 100-150 Pa，符合燃机运行要求。
2.  **寿命保障**：较低的风速能显著降低滤材表面的粉尘堆积速度。
3.  **结构合理性**：计算出的滤芯数量能使进气墙尺寸保持在合理的工程范围内。

## 4. 动态修正逻辑
系统会根据以下因素自动微调建议数量：
*   **海拔修正**：海拔升高 -> 空气变稀薄 -> 体积流量 $Q_{site}$ 增大 -> 滤芯数量增加。
*   **温度修正**：温度升高 -> 空气密度减小 -> 体积流量 $Q_{site}$ 增大 -> 滤芯数量增加。
*   **环境工况**：在沙漠或高尘环境，系统会建议增加预过滤级数，以保护终滤芯。
