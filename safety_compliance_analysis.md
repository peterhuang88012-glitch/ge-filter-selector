# 防爆门（Blow-in Doors）安全参数深度分析与合规性建议

## 1. 参数物理意义与工程取值依据

### 1.1 安全余量系数 (k)
- **物理意义**：代表了系统对“误开启”与“延迟开启”之间的容忍度。它本质上是系统可靠性与结构安全性之间的平衡因子。
- **工程取值依据**：
    - **低取值 (1.1-1.2)**：适用于环境极度恶劣（如极寒结冰、突发沙尘暴）的地区。物理上，这些工况下压降上升斜率极陡，较小的 $k$ 值能确保在进气道结构受损前瞬间开启。
    - **高取值 (1.4-1.5)**：适用于标准城市工况。物理上，压降上升较慢，较高的 $k$ 值能有效防止因燃机负荷剧烈波动（如甩负荷）产生的瞬时负压导致防爆门误开，从而避免未过滤空气进入燃机。

### 1.2 系统设计最大压降 (DP_max)
- **物理意义**：这是进气系统结构设计的“红线”。它代表了进气道壁板在不发生永久性塑性变形的情况下所能承受的最大静压差。
- **工程取值依据**：通常由进气道结构强度决定。在工程实践中，我们取滤芯达到“报废压降”（Terminal DP）时的数值。GMRC 指南建议该值应覆盖所有过滤级数的总和再加上 20% 的结构余量。

### 1.3 空气密度修正 (ρ_site / ρ_ISO)
- **物理意义**：修正了流体动能对压力的贡献。在相同流速下，密度越大，产生的压差力越大。
- **工程取值依据**：必须基于现场最恶劣的极端低温（密度最大）和最高海拔进行校核。

## 2. 行业标准对比分析

| 维度 | 本系统计算结果 (v8.0) | GMRC 指南推荐 | 典型 OEM 标准 (GE/Siemens) |
| :--- | :--- | :--- | :--- |
| **开启压力设定** | 110 - 150 mmH2O | 100 - 150 mmH2O | 125 - 175 mmH2O |
| **安全余量 (k)** | 1.2 - 1.5 | 1.25 (典型值) | 1.3 - 1.6 |
| **响应时间** | 瞬时机械开启 | 建议 < 1s | 必须为无源机械触发 |
| **环境修正** | 动态密度修正 | 建议考虑海拔 | 强制要求考虑极端低温 |

**分析结论**：本系统的计算逻辑与 GMRC 指南高度一致，但在安全余量 $k$ 的取值上比部分 OEM 标准略显保守（更敏感），这更有利于保护进气道结构安全。

---

## 3. 最终合规性建议

1.  **设定值合规性**：建议将防爆门开启压力设定在 **125 mmH2O (1225 Pa)** 左右作为通用基准。如果现场属于极寒地区，应下调至 **110 mmH2O**。
2.  **硬件合规性**：防爆门必须采用**无源机械式**结构（如重锤式或弹簧式），严禁使用依赖电力驱动的电动阀作为防爆门。
3.  **监控合规性**：必须在防爆门上安装**限位开关**，并将信号接入燃机保护系统（Trip Logic）。一旦开启，燃机必须立即跳机或强制减载。
4.  **维护合规性**：根据 ISO 21789 要求，防爆门应纳入年度强制检修清单，检查密封条是否老化以及机械结构是否卡涩。
